# Tarefa Módulo 2

## Aluno: Rodrigo Pucci Flores

Renderização de cubos em diferentes posições, com controle no teclado para rotação, translação e escala.

Controles:
- A: Esquerda
- D: Direita
- W: Para trás (não há projeção perspectiva)
- S: Para frente
- I: Para cima
- J: Para baixo
- \[: Aumentar
- \]: Diminuir (´ teclado ABNT)
